# Comparative Analysis: MobileNetV2 vs NASNetMobile for Multi-Task Age Group & Gender Prediction

This repository contains the GUI/frontend for a school project. It uses mock prediction data for now so the interface can be built and tested before the machine learning models are integrated.

## Features
- Image upload and preview
- Model selector for MobileNetV2 and NASNetMobile
- Mock prediction button
- Results panel for gender, age group, confidence, and selected model

## Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the app:
   ```bash
   python app.py
   ```
